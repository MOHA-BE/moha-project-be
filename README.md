# :hearts: Moha_MiniProject


전국 각지의 간단한 숙박업소 리뷰사이트 입니다.

항상 숙소 검색과 후기 찾으시느라 어려움을 느끼시는 분들을 위한 서비스!!
각 지역의 숙소 이용객들이 올려 놓은 후기로 정보를 얻고
내가 가본 좋은 숙소 후기를 올려 정보를 공유 할 수 있습니다.
